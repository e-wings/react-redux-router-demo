React Redux Router Form Demo

Another redux demo with no router: react-redux-demo

This is a react-redux version Counter, integrated redux-router, redux-form, and based on gulp, stylus and webpack. The internal data flow is handled with react-edux .

More info, more function about redux-form: redux-form

Installation

$ git clone https://github.com/e-wings/react-redux-router-demo.git
$ cd react-redux-router-demo
$ npm install
$ gulp
